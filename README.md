# Network Intrusion Detection System (IDS)

A machine learning project designed to identify and classify network traffic as either 'Normal' or 'Attack' using a Decision Tree classifier.

## Project Overview
This system simulates network traffic and employs a machine learning model to detect potential security threats. The project demonstrates the full data science lifecycle, from synthetic data generation to model training and evaluation.

## Key Features
- **Data Generation:** Created a synthetic dataset simulating network traffic features (e.g., protocol type, service, byte counts).
- **Data Preprocessing:** Utilized `LabelEncoder` to transform categorical network features into numerical data suitable for machine learning.
- **Model Training:** Implemented a `DecisionTreeClassifier` to learn from network patterns.
- **Model Evaluation:** Achieved 100% classification accuracy on the training dataset.

## Technologies Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn
- **Environment:** Jupyter Notebook

## How to Run
1. Ensure you have Python and Jupyter installed.
2. Clone this repository.
3. Open `Untitled1.ipynb` in your Jupyter environment.
4. Run the cells sequentially to observe data generation, model training, and predictions.
